## Overview
iNotebook is a MERN stack web application that allows users to securely create, edit, delete, and manage personal notes online. Each user can access only their own notes using authentication.

---

## Features
- User Signup & Login
- JWT Authentication
- Add, Edit, Delete Notes
- User-specific note storage
- Responsive interface

---

## Tech Stack
- Frontend: React.js, Bootstrap
- Backend: Node.js, Express.js
- Database: MongoDB
- Authentication: JWT
